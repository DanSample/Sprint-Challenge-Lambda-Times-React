1. PropTypes are use to validate data being fed into a component. The reason it is important to check our data, is to ensure that the component is receiving the data type it is expecting.

2. A life-cycle event in React is a method that is called during the life-cycle of a component. The 'life-cycle' has 3 phases; Mounting, Updating, and Un-mounting.

3. A Higher Order Component is a component(function) that takes a component and returns a new component. It is a technique used for reusing a components logic on the newly created components. 

4. You can use inline CSS, this way you don't have specificity issues. Libraries like react-strap, which give you pre-styled components that usually have many options to adjust the look of those components. Libraries like styled-components, which let you write css in javascript.
